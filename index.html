AI-Powered Call System / AI Customer Service App

Oct 2025 – Dec 2025
This project is a real-time AI-driven call system designed to handle customer support calls efficiently. It listens to customer speech, transcribes it, identifies intent using AI, and routes callers to the appropriate department automatically. The system improves response time, reduces human intervention, and ensures a seamless customer experience.

Key Features

  • Real-time voice transcription using Whisper AI
  • Intent detection with LLM integration to route calls to Billing, Technical Support, Sales, etc..
  • Audio buffering and post-call processing for accurate transcription.
  • Scalable backend using FastAPI and PostgreSQL to handle multiple concurrent calls.

End-to-end workflow with tools & integration:

  • Incoming Call: Twilio or Media Streams captures live audio.
  • Audio Streaming & Preprocessing: μ-law audio decoded to PCM → buffered in FastAPI.
  • Post-Call Audio Handling: Full conversation stored as WAV after call ends.
  • Transcription: Whisper AI converts audio to text.
  • Intent Analysis: Text sent to LLM → determines caller intent
  • Routing: Based on intent, call is directed to the correct department; loops back if intent is unclear.
  • Database Storage: PostgreSQL logs call transcripts, intent, and metadata for reporting.
  • Frontend Dashboard : Admins can monitor calls and transcripts in real-time.

Job Application Agent

Oct 2025 – Dec 2025
The Job Application Agent is an AI-powered system designed to simplify and accelerate the job application process. It understands a user’s resume and profile, analyzes job descriptions, and intelligently matches skills and experience to generate tailored applications. The goal is to reduce repetitive manual work, improve application quality, and help candidates apply more efficiently and strategically.

Key Features

  • Intelligent resume and profile understanding using LLMs.
  • Automated analysis of job descriptions to extract required skills and qualifications.
  • Resume customization and content optimization for specific roles.
  • Interactive chatbot interface to guide users through the application process.

End-to-End Workflow with Tools & Integration

  • Starting Node – User Upload: Users upload their resume through the website interface.
  • FastAPI Preprocessing: Backend pre-processes the resume and profile data for analysis.
  • LLM Node – Ollama Model: Processed data is sent to the Ollama model to identify matching job roles.
  • Job API Integration: Matching job metadata is fetched via a free jobs API (includes job info and application links).
  • User Display: Results are displayed on the user dashboard for direct application.
  • Chatbot Page: AI-powered assistant provides guidance, application tips, and answers queries.
  • LangGraph Orchestration: Connects all nodes, ensuring proper workflow and data flow between steps.
